# Animal Health Advisory System

Welcome to the Animal Health Advisory System! This website is designed to provide valuable information and resources related to animal health and welfare.

## Features
- **Disease Information:** Comprehensive details about various animal diseases, symptoms, and treatment options.
- **Veterinary Directory:** A list of certified veterinarians and animal healthcare facilities.
- **Health Tips:** Expert advice on animal care, nutrition, and preventive measures.
- **Discussion Forum with veterinarians:** A platform for users to share experiences, ask questions, and seek advice from the community.
- **Get new events and newss related to animal health and resources related to animal to animal health and.

## How to Use
1. **Navigate:** Use the menu to explore different sections of the website.


## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** mysql

## Installation
To run the website locally, follow these steps:

1. Clone the repository:
    ```bash 
    git clone https://github.com/adane-desta/animal-health-advisory-system.git
    ```
2. Navigate to the project directory:
    ```bash
    cd animal-health-advisory-system
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the server:
    ```bash
    npm start
    ```
